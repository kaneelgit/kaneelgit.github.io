Training agents tic-tac-toe using Reinforcement Learning

Here we use Reinforcement Learning (RL) to teach an agent to play tic-tac-toe. The RL agent is playing games with another RL agent and a random agent and learn to master the game from past experience. The results show that the RL agent has a significant advantage over the random agent. Also when playing against another RL agent both agents learn to draw games in order to maximize their rewards.

Audio-Visual Integration



In this jupyter notebook, I have recreated the audio-visual integration model from the paper "The Ventriloquist Effect Results from Near-Optimal Biomodal Integration". The paper uses a Bayesian model to describe how humans optimally integrate vision and audition.

Visuoproprioceptive Integration



In this jupyter notebook, I have created a similar simulation to the one in the paper "Integration of Proprioceptive and Visual Position: An Experimentally Supported Model" The paper uses a 2 dimensional Bayesian model to describe how humans optimally integrate vision and proprioception. Interestingly, the paper shows that the brain integrate the two modalities based on the directional precision of each sense.
